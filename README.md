# OnlineShopWeb

OnlineShopWeb is a React-based web application that allows users to browse, sort, and search through a catalog of products. Additionally, you can add new products using a simple form interface.

## Features

- View a list/grid of products
- Search products by name
- Sort products by price
- Add new products through a form

## Technologies Used

- React
- TypeScript
- Vite
- Material-UI (MUI)
- ESLint
